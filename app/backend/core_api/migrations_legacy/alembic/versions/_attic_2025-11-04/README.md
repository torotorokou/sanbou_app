このディレクトリには、Alembic 運用の再構築に伴ってアーカイブされた過去のリビジョンファイルが含まれています。

## アーカイブ理由

- **DB未刻印**: これらのリビジョンは `alembic_version` テーブルにまだ刻印されていない（未適用）状態でした
- **単一ベースライン方針**: 新運用では単一のベースライン (`9a092c4a1fcf_baseline_no_op.py`) から再構築するため、これらは使用しません
- **保全目的**: 削除せずアーカイブとして残すことで、過去の設計意図や変更履歴を参照・検証可能にしています
