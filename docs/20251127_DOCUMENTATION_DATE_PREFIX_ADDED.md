## メリット

1. **時系列把握**: ファイル名だけで作成日が分かる
2. **ソート容易**: ファイル名でソートすると時系列順になる
3. **検索効率**: 特定期間のドキュメントを素早く特定可能
4. **履歴管理**: プロジェクトの進化過程を視覚的に把握

## 注意事項

- README.mdファイルは対象外（常に最新を保つため）
- ファイル名に既に日付が含まれていた場合も、先頭に作成日を追加
- 日付はファイルの最終更新日（stat -c %y）から取得
