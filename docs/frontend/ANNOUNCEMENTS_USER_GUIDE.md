# お知らせ機能 - ユーザーガイド

**対象**: 社内ポータルシステムのエンドユーザー

---

## 📢 お知らせ機能とは

社内ポータルシステムの「お知らせ」機能は、システムの更新情報、メンテナンス通知、重要な社内連絡などを受け取るための機能です。

---

## 🎯 主な機能

### 1. トップページの重要通知バナー

トップページ（ポータル画面）にアクセスすると、重要なお知らせがバナーで表示されます。

**特徴**:
- 赤色（重要）またはオレンジ色（注意）で表示
- 「理解しました」ボタンをクリックすると、次回から表示されません
- 「×」ボタンでも同様に非表示にできます

**例**:
```
⚠️ システムメンテナンスのお知らせ
12月28日（土）02:00〜05:00にシステムメンテナンスを実施します...
[理解しました] ✕
```

---

### 2. お知らせ一覧ページ

すべてのお知らせを一覧で確認できます。

**アクセス方法**:
1. 画面左のサイドバーから「ホーム」→「お知らせ」をクリック
2. または、トップページの「お知らせ」カードをクリック

**画面の見方**:

| 要素 | 説明 |
|------|------|
| 🔵 青色バッジ | 情報レベル（通常のお知らせ） |
| 🟠 オレンジ色バッジ | 注意レベル（要確認） |
| 🔴 赤色バッジ | 重要レベル（必読） |
| 🟣 紫色バッジ | ピン留め（重要度が高い） |
| 緑色の背景 + 左ボーダー | 未読のお知らせ |
| 通常の背景 | 既読のお知らせ |

**操作方法**:
1. 一覧からお知らせをクリック
2. 詳細が表示されます（クリックした時点で「既読」になります）
3. 詳細画面を閉じると、一覧に戻ります

---

### 3. 未読バッジ

サイドバーの「お知らせ」メニューに、未読のお知らせ数がバッジで表示されます。

**例**:
```
ホーム ▼
  ├─ トップページ
  └─ お知らせ [2] ← 未読が2件
```

**動作**:
- お知らせを開くと、自動的に既読になります
- 既読になると、バッジの数字が減ります
- すべて既読になると、バッジが消えます

## 📖 よくある質問（FAQ）

### Q1. お知らせが表示されません

**回答**:
- 現在、公開期間内のお知らせがない可能性があります
- ブラウザをリロード（F5）してみてください

### Q2. 一度閉じたバナーをもう一度見たい

**回答**:
サイドバーの「お知らせ」から一覧ページにアクセスすれば、すべてのお知らせを確認できます。

### Q3. 未読・既読の状態はどこに保存されますか？

**回答**:
現在は、お使いのブラウザに保存されています。  
別のブラウザや端末では、未読・既読の状態がリセットされます。

※ 将来的には、ユーザーアカウントに紐付けて管理する予定です。

### Q4. お知らせを検索できますか？

**回答**:
現在、検索機能はありません。  
一覧ページで上下にスクロールして確認してください。

### Q5. 重要なお知らせを見逃さないようにするには？

**回答**:
- トップページに赤色またはオレンジ色のバナーが表示された場合は、必ず内容を確認してください
- サイドバーの未読バッジを定期的にチェックしてください

## 💡 活用のヒント

### 定期的な確認を習慣に

- **毎朝**: トップページにアクセスして、重要通知を確認
- **週1回**: お知らせ一覧で、未読のお知らせをチェック

### 重要度の見分け方

| 色 | 重要度 | 対応 |
|----|--------|------|
| 🔴 赤色 | 必読 | すぐに内容を確認し、必要な対応を行う |
| 🟠 オレンジ色 | 要注意 | 時間のあるときに確認する |
| 🔵 青色 | 情報 | 参考として確認する |
